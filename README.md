# 🍅 Pomodoro Timer

Приложение для управления временем по методу Помодоро. Написано на Python + PyQt5.

## ✨ Возможности

- **Таймер** — визуальный круговой отсчёт времени
- **Гибкая настройка** — выбор длительности сессии от 1 до 120 минут
- **🔥 Streak** — счётчик дней использования. Растёт если открываешь приложение каждые 24 часа
- **📝 To-Do лист** — задачи сохраняются между запусками, можно отмечать галочкой

## 🛠 Стек

- **Python 3.15**
- **PyQt5** — GUI
- **SQLite** — хранение задач и стрика (без сервера, просто файлы)

## 🚀 Запуск

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Запусти приложение

```bash
python main.py
```

## 📁 Структура проекта

```
PomodoroTimer/
├── main.py          # GUI приложение
├── db.py            # Работа с базой данных
├── requirements.txt # Зависимости
├── data.db          # БД задач (создаётся автоматически)
└── streak.db        # БД стрика (создаётся автоматически)
```

## 🗄 База данных

Проект использует два SQLite файла которые создаются автоматически при первом запуске.

**data.db** — хранит to-do задачи:
```
id | text       | done
1  | buy milk   | 0
2  | study      | 1
```

**streak.db** — хранит стрик:
```
id | last_used            | streak_count
1  | 2026-03-05 10:00:00  | 3
```
